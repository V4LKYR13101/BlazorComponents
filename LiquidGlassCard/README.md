#	Liquid Glass Card (IOS 26 Inspired)

This usage an SVG displacement filter to create a liquid glass effect on a card component. SImple but effective.

## Usage
```html
<LiquidGlassCard>
    <BodyContent>
        <h2>Title</h2>
        <p>Description</p>
    </BodyContent>
</LiquidGlassCard>
```
